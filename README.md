#SKULBUNY.COM

###What is this?
This is the source code that powers [skulbuny.com](http://skulbuny.com).

###Requirements
- Ruby 1.9.3
    `rvm install Ruby 1.9.3`
- Rubygems
    `curl -L get.rvm.io | bash -s stable`
- Jekyll 1.2.1
    `gem install jekyll --pre`
- Rake
    `gem install rake`
- Anemone
    `gem install anemone`
